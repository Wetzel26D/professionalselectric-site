Panel FAQ

Before you schedule.

Do I need a panel upgrade for an EV charger?

Not always. The existing panel and load need to be reviewed first so the safest option can be confirmed.

What photos should I send?

Send the full panel, the breaker labels, the main breaker area, and the location where the new circuit or equipment would go.

Can you add dedicated circuits?

Yes. Dedicated circuits can support appliances, tools, EV prep, garage equipment, and other higher-use loads.
